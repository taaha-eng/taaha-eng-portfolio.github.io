Drop your capstone media into this folder using these exact filenames,
and they'll appear automatically on the site (no code changes needed):

  capstone-proboscis-render.jpg     -> CAD render of the proboscis mechanism
  capstone-drone-drawing.jpg        -> Technical drawing of the drone assembly
  capstone-pollination-animation.gif -> Motion animation of the pollination cycle
                                        (a short .mp4 works too, but you'd need to
                                        swap the <img> tag for a <video> tag in
                                        index.html if so — ask Claude to do this)

If a file is missing, the placeholder frame still displays gracefully
(no broken-image icon) so it's safe to launch before all media is ready.

Recommended size: 1200px wide max, JPG for photos/renders, GIF or MP4 for
the animation. Keep file sizes under ~2MB each for fast loading.

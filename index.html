SHEET 02Featured Project

PROJECT NO. 001

Automated Pollination Drone

Greenhouse Automation — Capstone Project

DOCUMENTED

Problem

Pollinating greenhouse crops by hand or by managing live pollinator insects is labor-intensive and difficult to control consistently in an enclosed environment. This project explored automating that contact using a drone-mounted mechanism.

My Role

I was responsible for the mechanical design of the proboscis — the actuated component that makes contact with the flower and transfers pollen. This included the CAD model and the motion animation used to validate reach and contact behavior.
